# SUN CLI

SUN CLI (`@bankofai/sun-cli`) is a command-line tool that brings the full power of [SUN.IO (SunSwap)](https://sun.io) to your terminal. Whether you operate it by hand or drive it from an AI agent, it lets you query DeFi data, swap tokens, and manage liquidity on TRON through a single command-line interface.

Where SUN MCP Server exposes these capabilities as MCP tools for AI assistants to call natively, SUN CLI wraps the same capabilities as structured commands for manual use, scripting, or shell invocation by agents.
