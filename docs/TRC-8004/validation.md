# Validation
## What is the Validation Registry for?
To cryptographically validate the behavior of agents. Validation logic is delegated to different smart contracts. Some approaches we are working on with the community include:


- Security based on attestations from Trusted Execution Environments
- Crypto-economic security (staking validators re-execute the inference)
- zkML




## Why isn't it available yet?
A technical discussion is still ongoing with the community. As soon as the outcome is solid and shared by the majority of involved stakeholders, we will finalize that section of the specs and go live.
