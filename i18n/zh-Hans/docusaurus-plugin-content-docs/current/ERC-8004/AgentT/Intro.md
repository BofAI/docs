# 简介

AgentT 是 **ERC-8004 标准的参考实现**，专为 Agentic Economy 打造。作为一套 SDK，它利用区块链与去中心化存储赋予 Agents 注册身份、发布能力及建立信誉系统的功能，从而实现了无需依赖中介的无许可发现机制。


## AgentT 能做什么？

AgentT SDK 的核心功能包括：

### 1. 创建与管理智能体身份
* 在链上注册 AI 智能体并生成唯一身份。
* 配置展示字段（如名称、简介、头像）。
* 设置钱包地址，并管理支持 **x402 协议**的信任模型。

### 2. 发布 Agents 能力
* 发布 MCP（Model Context Protocol）和 A2A（Agent-to-Agent）端点。
* SDK 支持从端点中自动提取 MCP 工具集和 A2A 技能信息。

### 3. OASF 分类体系支持
* 利用 **“开放 Agents 模式框架”** 来发布标准化的技能与领域信息，从而显著提升智能体的可发现性。

### 4. Agnets 查询与发现
* 支持通过 `chainId:agentId` 格式或默认链设置，查询当前支持网络上的智能体。
### 5. 实现无许可发现：
* 让其他智能体和平台能够通过属性、能力、技能、工具、任务及 x402 支持情况等丰富维度轻松发现您的智能体。

### 6. 构建信誉系统
* 支持发送和接收反馈。
* 检索反馈历史记录。
* 允许基于加密认证搜索具有特定信誉等级的智能体。

### 7. 公共索引
* 利用 **Subgraph（子图）** 技术同时对链上数据和 IPFS 数据进行索引，实现极速的搜索与检索体验。
