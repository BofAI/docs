# Grok 4.5

## 概述

Grok 4.5 是 SpaceXAI 的推理模型，适用于编程、智能体任务、工程和知识工作。在 B.AI 中使用模型 ID：`grok-4.5`。

## 核心特性

* **智能体工程：** 适用于多步骤编程、调试、重构和仓库级工作。
* **可配置推理：** 支持推理强度控制，用于平衡响应速度和推理深度。
* **长多模态上下文：** 支持文本和图片输入，上下文窗口最多 500,000 个 token，输出为文本。
* **工具调用工作流：** 支持函数调用和结构化输出；额外工具能力取决于 API 配置。
* **长上下文定价：** 提示词超过 200,000 个 token 的请求采用长上下文价格档位。

## 推荐场景

* 仓库级编程，包括实现、调试、重构和技术审查。
* 结合结构化输出、函数调用和重复推理循环的工具调用智能体。
* 面向工程、科学、数学以及混合文本/图片材料的技术分析。
* 长上下文文档和知识工作流。
