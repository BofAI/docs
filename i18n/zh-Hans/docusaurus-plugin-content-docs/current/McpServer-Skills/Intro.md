# 简介

Bank of AI 通过 MCP (模型上下文协议) 与 Skills (技能库) 的双层架构，为构建基于区块链网络的自主智能体（AI Agents）提供了从基础设施 to 业务逻辑的完整解决方案。

**基础设施层**：mcp-server 作为智能体的“手和眼”，MCP 服务器利用标准化的协议将区块链能力无缝集成到 AI 中。它为智能体提供了一个统一的交互界面，使其能够自主读取区块数据、管理钱包（支持 Hex/Base58 格式）、执行转账，并与任意智能合约进行读写交互，全面覆盖主网及测试网。

**应用逻辑层**：skills 作为智能体的“大脑与操作手册”，Skills 库提供了一系列可复用的能力模块。它通过结构化的指令文档（SKILL.md）教会 AI 如何组合调用 MCP 提供的底层工具来完成复杂的业务流程。

**DeFi 交互**：通过 SunSwap 技能库实现代币自动兑换，未来计划支持JustLend，pancake，lista。

**身份与信誉**：利用 ERC-8004 标准管理链上身份注册与信誉积累。

**经济支付**：集成 x402 协议，赋予智能体自动化的收付款能力。

## 总结
开发者只需部署 MCP 服务器并加载相应的 Skills，即可让 AI 智能体从“聊天机器人”进化为具备资产管理能力与链上交互能力的 Web3 经济实体。
