# SUN CLI

SUN CLI 是一个命令行工具，把 [SUN.IO（SunSwap）](https://sun.io)的完整能力带到你的终端。无论是手动操作还是交给 AI Agent 驱动，它都让你通过单一命令行接口在 TRON 上查询 DeFi 数据、兑换代币、管理流动性。

SUN MCP Server 把这些能力以 MCP 工具的形式暴露给 AI 助手原生调用；SUN CLI 则把同样的能力封装成结构化命令，供手动使用、脚本编排，或由 Agent 通过 shell 调用。
