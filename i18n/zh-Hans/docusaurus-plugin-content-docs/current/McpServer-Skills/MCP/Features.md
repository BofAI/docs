# 功能

### 区块链数据访问

*   **TRON 网络支持**：主网、测试网。
*   **链信息**：区块号、链 ID、RPC 端点。
*   **区块数据**：按编号或哈希访问。
*   **交易详情**：详细信息，包括资源使用（能量/带宽）。
*   **资源成本**：查询当前链参数以获取能量和带宽价格。

### 代币服务

*   **原生 TRX/BNB**：检查余额和转账。
*   **TRC20/BEP-20 代币**：
    *   检查余额。
    *   转移代币。
    *   获取代币元数据（名称、符号、小数位数）。

### 地址服务

*   **格式转换**：在 Hex (`41...` 或 `0x...`) 和 Base58 (`T...`) 格式之间转换。
*   **验证**：验证地址在相应的链上是否有效。

### 智能合约交互

*   **读取合约**：调用 `view` 和 `pure` 函数。
*   **写入合约**：执行改变状态的函数。
*   **ABI 获取**：自动从区块链获取已验证合约的 ABI。

### 钱包与安全

*   **灵活钱包**：通过 `PRIVATE_KEY` 或 `MNEMONIC` 配置。
*   **HD 钱包**：支持 BIP-44 派生路径 `m/44'/195'/0'/0/{index}`。
*   **签名**：签署任意消息。

## 支持的网络

*   **TRON 主网**：`mainnet`
*   **Nile 测试网**：`nile`
*   **Shasta 测试网**：`shasta`
*   **BSC 主网**：`mainnet`
*   **Chapel 测试网**：`chapel`
