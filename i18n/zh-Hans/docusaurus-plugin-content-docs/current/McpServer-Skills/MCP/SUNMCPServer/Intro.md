# SUN MCP Server

SUN MCP Server 是连接 AI 助手与 [SUN.IO（SunSwap）](https://sun.io)的桥梁——SunSwap 是 TRON 上最大的去中心化交易所。它基于 Model Context Protocol（MCP）构建，让你用自然语言查询 DeFi 数据、执行代币兑换、管理流动性头寸，无需直接接触合约或链上参数。
